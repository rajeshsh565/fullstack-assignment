## Setup Instructions

1. **Setup .env for the `backend`**
    - Create .env file in `backend` folder.
    - Add a mongodb uri as follows -
        ```
        MONGODB_URI = uri_goes_here
        ```

2. **Install concurrently (dependency)**
   ```
   npm install
   ```

3. **Run the following cli command on root folder `fullstack-assignment`**
   ```
   npm run dev
   ```

4. **Add a few cards in you database using backend APIs in Postman or ThunderClient otherwise, cards section will be empty**

5. **Open the Project in any Browser**
    ```
    http://localhost:5173/
    ```
